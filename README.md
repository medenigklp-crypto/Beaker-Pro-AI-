
# Beaker Pro AI

Modern, açık kaynaklı, çoklu yapay zekâ destekli masaüstü çalışma ortamı.

## Özellikler

- Modern PySide6 arayüzü
- Çoklu AI sağlayıcısı
- PDF / Word / Excel analizi
- OCR
- Speech To Text
- Text To Speech
- Plugin Sistemi
- Workflow Builder
- AI Agent
- Memory
- RAG
- Local LLM
- Ollama
- OpenAI
- Gemini
- Claude

## Kurulum

```bash
git clone https://github.com/USERNAME/Beaker-Pro-AI.git

cd Beaker-Pro-AI

pip install -r requirements.txt

python app.py
```
